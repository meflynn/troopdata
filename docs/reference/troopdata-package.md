# Tools for Analyzing Cross-National Military Deployment and Basing Data

Generate customized data frames containing U.S. military deployment data

## Details

The `troopdata` package contains functions for generating customized
data sets on U.S. military deployment data and U.S. overseas basing
data. The data contain country identifiers that can be used to merge
these data with other data to analyze trends in the United States'
overseas military posture.
